## 🔴 গুরুতর সমস্যা (Critical)

### 1. `src/middleware/authMiddleware.ts` — অথেনটিকেশন দুর্বল
**সমস্যা**: কোনো API key validation নেই। যেকোনো র‍্যান্ডম স্ট্রিং `apiKey` হিসেবে সেট হয়ে যাচ্ছে।
**ঝুঁকি**: `GROQ_API_KEY` না থাকলে `apiKey = undefined` সেট হয় — 500 এরর দেখায় না।
**সমাধান**: API key না থাকলে 500 রিটার্ন করুন। ফাঁকা `Bearer ` পাঠালে 401 দিন।

### 2. `src/controllers/dbController.ts` — SQL Injection ঝুঁকি
**সমস্যা**: `handleDbList`-এ `req.params.table` সরাসরি SQL query-তে পাস হয় — কোনো whitelist নেই।
**ঝুঁকি**: যেকোনো table name দিয়ে সব data পড়া যায় (e.g., `/db/identity`)।
**সমাধান**: Whitelist table names যোগ করুন।

### 3. `src/services/identityService.ts` — chmod Security Bug
**সমস্যা**: `fs.chmodSync(ID_PATH, 0o444)` — ফাইলকে permanent read-only করে দিচ্ছে।
**ঝুঁকি**: পরবর্তীতে identity update করলে overwrite হবে না।
**সমাধান**: `chmod` বাদ দিন।

### 4. `src/controllers/openaiController.ts` — Dashboard XSS
**সমস্যা**: `handleDashboard`-তে user-controlled data (model.id, identityName) সরাসরি HTML-এ inject।
**ঝুঁকি**: Reflected XSS vulnerability।
**সমাধান**: `escapeHtml()` utility ব্যবহার করুন।

### 5. `src/routes/index.ts` — Auth Bypass
**সমস্যা**: `/db/*` routes authenticate middleware-তে wrap করা নেই।
**ঝুঁকি**: যেকোনো `/db/identity` URL দিয়ে identity data চুরি করা যায়।
**সমাধান**: `authenticate` middleware-তে `/db` routes যোগ করুন।

### 6. `src/services/stateDb.ts` — Type Safety বাতিল
**সমস্যা**: `export type StateDb = any` — better-sqlite3 Database type ব্যবহার করা উচিত।
**ঝুঁকি**: `upsertIdentity`-তে SQL injection risk।
**সমাধান**: Proper types ব্যবহার করুন।

## 🟡 মাঝারি সমস্যা (Medium)

### 7. `src/services/groqService.ts` — Memory Leak
**সমস্যা**: `rateCounters` Map — cleanup mechanism নেই। `estimateTokens()` ভুল।
**সমাধান**: Rate counter-এ timer-based cleanup দিন।

### 8. `src/services/ragService.ts` — Sync I/O
**সমস্যা**: `scanProject()` — synchronous `readdirSync` — event loop block।
**সমাধান**: Async file I/O ব্যবহার করুন।

### 9. `src/controllers/agentController.ts` — SRP Violation
**সমস্যা**: `handleAgentChat` — 190 লাইনের function। Plaintext tokens।
**সমাধান**: Smaller functions-এ ভাগ করুন।

### 10. `src/middleware/loggingMiddleware.ts` — Dead Code
**সমস্যা**: `(res as any).__body` কখনো ব্যবহার হয়নি। Double logging।
**সমাধান**: `res.on('finish')` ব্যবহার করুন।

### 11. `src/middleware/identityMiddleware.ts` — Info Leak
**সমস্যা**: প্রতিটি রেসপন্সে `X-Powered-By` পাঠাচ্ছে।
**সমাধান**: Production-এ এই headers বাদ দিন।

### 12. `src/controllers/mcpController.ts` — Memory Leak
**সমস্যা**: `mcpSessions` Map — SSE response reference ধরে রাখে।
**সমাধান**: Session cleanup improve করুন।

### 13. `identity.json` — Personal Data Exposed
**সমস্যা**: ব্যক্তিগত তথ্য (ফোন, ঠিকানা) JSON-এ।
**সমাধান**: ব্যক্তিগত তথ্য `.env`-তে রাখুন।
